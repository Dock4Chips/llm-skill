# Metrics

## GitHub Sections

- `每日 Star 最多的仓库`: repositories ranked by star count from the configured query.
- `每日 Fork 最多的仓库`: repositories ranked by fork count from the configured query.
- `每日趋势榜`: repositories scraped from GitHub Trending.
- `人工智能方向热榜`: category heading for AI-related subsections.
- `Python`: currently implemented AI subsection.

## Normalization

- `title`: repository or item name shown to the user.
- `url`: canonical item link.
- `description`: short plain-text summary.
- `language`: programming language or `N/A`.
- `stars`: stringified star count.
- `forks`: stringified fork count.
