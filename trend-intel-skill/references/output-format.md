# Output Format

Every report should be Markdown and follow this shape:

```md
# <Platform Name> <date>

## 目录

- section links

## <Section A>

| 序号 | 名称 | 描述 | 语言 | Stars | Forks |
| --- | --- | --- | --- | --- | --- |
| ... |
```

## Rules

- Keep headings stable.
- Prefer tables for ranked results.
- Escape `|` in descriptions.
- Keep descriptions short enough for readable tables.
- If a section has no data, write `*暂无数据*`.
